How it works

A memory fabric, not another cloud subscription.

Agents drop a file into a per-project inbox. The daemon weaves every finding, decision, and code change into one live knowledge graph — visible and governable from a dashboard.

⌥

Filesystem inbox

Any agent connects by writing a file. Zero SDK, zero auth, zero API keys. The simplest possible integration surface.

◰

Single-process daemon

pip install and run. No Docker, no Neo4j, no external database, no cloud account.

⌗

Code-aware graph

Parses your codebase AST into files, functions, classes, and call flows — then enriches it with what agents learn.

◳

Dashboard control plane

Browse the graph, watch agents live, search across everything, and dispatch work — all from one UI.

⇄

MCP + hybrid search

An MCP server any agent can query, plus graph + vector + relational search in a single call.

▣

Sandboxed task board

Dispatch tasks to agents that run in isolated git worktrees with a per-task budget cap. Review the diff, then merge.
